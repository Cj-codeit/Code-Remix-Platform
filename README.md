# Code Remix Platform

Distributed platform for students to hack, share, and remix coding projects.

## Overview

A blockchain-based platform that encourages collaborative learning through code sharing and remixing. Students can share their coding projects, discover others' work, create remixes with improvements, and build a portfolio of contributions.

## Features

- Developer profile management
- Project sharing with metadata
- Project remixing and attribution
- Upvoting system for quality content
- Collaboration tracking
- Transparent contribution history

## Smart Contract Functions

### Public Functions
- `create-developer-profile` - Create your developer profile
- `share-project` - Share a coding project with the community
- `remix-project` - Create a remix of an existing project
- `upvote-project` - Upvote projects you find valuable
- `add-collaborator` - Add collaborators to your project

### Read-only Functions
- `get-project` - Retrieve project details
- `get-remix` - View remix information
- `get-developer-profile` - Check developer stats
- `has-upvoted` - Verify if user upvoted a project
- `get-total-projects` - Total projects shared
- `get-total-remixes` - Total remixes created

## Getting Started

1. Create your developer profile
2. Share your coding projects with repository links
3. Discover and remix interesting projects
4. Collaborate with other student developers
5. Build your coding portfolio on-chain
